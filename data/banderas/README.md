## Licencia

Las banderas de las provincias del Ecuador generalmente están en **Dominio Público** o bajo licencia **Creative Commons**. 

Se recomienda verificar la licencia específica de cada archivo en Wikimedia Commons antes de usar en publicaciones comerciales.

Para uso académico/investigación (como este proyecto), el uso es libre con atribución apropiada.

## Nota Técnica

Wikimedia Commons bloquea las descargas automatizadas para proteger sus servidores. Por eso la descarga debe ser manual. Este es el comportamiento esperado y respeta sus términos de servicio.
